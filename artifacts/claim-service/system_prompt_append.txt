#### Step 3: Create Error Response and Page Response (MANDATORY)

**`app/schemas/common.py`:**
```python
from datetime import datetime
from typing import Generic, TypeVar

from pydantic import BaseModel

T = TypeVar("T")


class FieldError(BaseModel):
    field: str
    message: str


class ErrorResponse(BaseModel):
    status: int
    error: str
    message: str
    error_code: str
    details: list[FieldError] | None = None
    timestamp: datetime

    @classmethod
    def create(cls, status: int, error: str, message: str, error_code: str):
        return cls(status=status, error=error, message=message, error_code=error_code, timestamp=datetime.now())


class PageResponse(BaseModel, Generic[T]):
    content: list[T]
    page: int
    size: int
    total_elements: int
    total_pages: int
    first: bool
    last: bool

    @classmethod
    def create(cls, items: list[T], total: int, page: int, size: int):
        total_pages = (total + size - 1) // size if size > 0 else 0
        return cls(
            content=items,
            page=page,
            size=size,
            total_elements=total,
            total_pages=total_pages,
            first=page == 0,
            last=page >= total_pages - 1,
        )
```

**Insurance error codes:**
- `POL_NOT_FOUND` — Policy not found
- `POL_NOT_ACTIVE` — Policy not in ACTIVE status
- `POL_EXPIRED` — Policy expired
- `CLM_NOT_FOUND` — Claim not found
- `CLM_EXCEEDS_LIMIT` — Claim amount exceeds coverage limit
- `CLM_INVALID_DATE` — Date of loss outside policy period
- `COV_NOT_FOUND` — Coverage type not on policy
- `COV_LIMIT_EXCEEDED` — Coverage limit exceeded
- `UW_INELIGIBLE` — Insured does not meet eligibility criteria
- `QUO_EXPIRED` — Quote has expired (> 30 days)

from fastapi import APIRouter, Depends, Query, status
from sqlalchemy.orm import Session

from app.database import get_db
from app.schemas.policy import EndorsementRequest, CancelPolicyRequest, PolicyResponse
from app.schemas.common import PageResponse
from app.schemas.enums import PolicyStatus, PolicyType

router = APIRouter(prefix="/api/v1/policies", tags=["Policies"])


@router.get("", response_model=PageResponse[PolicyResponse],
            summary="List policies (paginated)")
def list_policies(
    page: int = Query(0, ge=0),
    size: int = Query(20, ge=1, le=100),
    status: PolicyStatus | None = None,
    policy_type: PolicyType | None = None,
    db: Session = Depends(get_db),
):
    """List all policies with pagination and optional filters."""
    return policy_service.find_all(db, page, size, status, policy_type)


@router.get("/{policy_no}", response_model=PolicyResponse,
            summary="Get policy details",
            responses={404: {"description": "Policy not found"}})
def get_policy(policy_no: str, db: Session = Depends(get_db)):
    """Get policy details by policy number."""
    return policy_service.get_by_number(db, policy_no)


@router.post("/{policy_no}/endorse", response_model=PolicyResponse,
             summary="Endorse (modify) an active policy",
             responses={400: {"description": "Invalid endorsement or policy not active"}})
def endorse_policy(policy_no: str, request: EndorsementRequest, db: Session = Depends(get_db)):
    """Apply an endorsement (modification) to an active policy."""
    return policy_service.endorse(db, policy_no, request)


@router.post("/{policy_no}/cancel", response_model=PolicyResponse,
             summary="Cancel a policy",
             responses={400: {"description": "Policy cannot be cancelled"}})
def cancel_policy(policy_no: str, request: CancelPolicyRequest, db: Session = Depends(get_db)):
    """Cancel an active policy."""
    return policy_service.cancel(db, policy_no, request)

             status_code=status.HTTP_201_CREATED,
             summary="Renew a policy for a new term")

## Conventions
- **Package Naming**: `app.{module}` structure with clear separation of concerns
- **API Prefix**: All endpoints use `/api/v1/` prefix for versioning
- **Error Format**: Consistent ErrorResponse schema with correlation IDs
- **Authentication**: JWT Bearer tokens in Authorization header
- **Pagination**: PageResponse envelope for all list endpoints with default 20, max 100 items
- **Logging**: Structured JSON in production, readable format in development
- **Data Masking**: SSN/TIN show only last 4 digits in all API responses
- **Type Safety**: Full type hints required, enforced by mypy
- **Async Patterns**: Use async/await throughout for database and external API calls
- **Transaction Safety**: Payment operations wrapped in database transactions
- **Audit Trail**: All data modifications tracked with user_id, correlation_id, timestamps

## Key Patterns
- **Dependency Injection**: Use FastAPI's Depends() for database sessions, auth
- **Repository Pattern**: Services layer abstracts database operations with async methods
- **Pydantic Models**: Separate request/response schemas from database models with validation
- **Async Context**: All database operations use async/await patterns with proper session management
- **Audit Trail**: Every data modification includes user_id and timestamp with comprehensive tracking
- **Security**: Sensitive data (SSN/TIN, payments) masked in responses and encrypted at rest
- **Error Responses**: Consistent error format with HTTPException and specific messages per requirements
- **Role-Based Access**: Decorator-based route protection by user role (admin, agent, adjuster)
- **Transaction Management**: Payment operations wrapped in database transactions
- **External Integrations**: Circuit breaker pattern with retry logic for resilience

- **Policy-Claim Relationships**: Foreign key relationships with claim-level policy overrides
- **Payment Complex Structures**: Multiple payees, reserve allocations, void/reversal chains
